Carrusel CRUD con funciones

Ejercicio:

Crear un carrusel de imágenes con las siguientes características: funcion avanza funcion retrocede funcion crearImagen funcion buscarImagen funcion modificarImagen funcion borrarImagen funcion salir funcion mostrarImagen Se deben usar funciones unicamente. Las imagenes estan dadas en la carpeta "./img". Se debe usar la variable posicion para saber en que posicion del array de imagenes se encuentra.

Pseudo codigo

1. Crear un array de objetos con las siguientes propiedades: image, name title, description 2. Crear una variable posicion para saber en que posicion del array de imagenes se encuentra, llamada "posicion" 3. Crear una funcion "avanza" que aumente en 1 la variable posicion 4. Crear una funcion "retrocede" que disminuya en 1 la variable posicion 5. Crear una funcion "crearImagen" que pida al usuario los datos con prompt y los guarde en el array de imagenes 7. Crear una funcion "modificarImagen" que pida al usuario el nombre de la imagen con prompt y la busque en el array de imagenes y modifique los datos de la imagen 8. Crear una funcion "borrarImagen" que pida al usuario el nombre de la imagen con prompt y la busque en el array de imagenes y la borre 9. Crear una funcion "salir" que salga del programa 10. Crear una variable que muestre el name de la imagen en el div con id "name" 11. Crear una variable que muestre el titulo del tema en el div con id "tema" 12. Crear una variable que muestre la descripcion de la imagen en el div con id "description"
